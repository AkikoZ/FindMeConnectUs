# FMCUClient - Main
这是 Find Me Connect Us 前端的游戏过程部分。

## 文件目录
* `assets` 静态资源目录
* `src` 源代码目录
* `package.json` npm 配置文件
* `webpack.config.js` webpack 配置文件

## 构建命令
``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080 (must build first for HTML changes!)
npm run start

# build for production
npm run build

# clean built files
npm run clean
```
